┌─────────────────────────────────────────────────────────────────────────────┐
│                     FORMAT PENJELASAN - STRUKTUR NOTA                        │
└─────────────────────────────────────────────────────────────────────────────┘

1. HEADER
   • Nama toko di CENTER (automatic padding)
   • Alamat toko di CENTER (optional)
   • Separator line (___ x 32 karakter)

2. TRANSACTION INFO (Left Aligned)
   • No.ID : [Invoice Number] (Right padded)
   • Tanggal : [Formatted Date] (Right padded)
   • Pelanggan : [Member/Regular] (Right padded)
   • [Customer Name] (Right padded)
   • Separator line

3. ITEMS
   • Product name (new line)
   • Qty x Price [Total] (right aligned harga)
   • Repeat untuk setiap produk

4. DISCOUNTS & POINTS (Conditional)
   • Diskon [Amount] (jika ada diskon)
   • poin di gunakan -[Qty] [Value] (jika ada poin)
   • Separator line

5. TOTALS
   • Subtotal [Amount] (Left label, Right amount)
   • TOTAL [Amount] (BOLD, Left label, Right amount)

6. PAYMENT INFO
   • Bayar [Amount]
   • Kembali [Amount]
   • metode [Payment Method]
   • poin +[Qty] (only untuk member)
   • Separator line

7. FOOTER
   • Message di CENTER (default: 'terima kasih sudah berbelanja')
   • Spacing untuk print cut

┌─────────────────────────────────────────────────────────────────────────────┐
│                         ALIGNMENT RULES                                      │
└─────────────────────────────────────────────────────────────────────────────┘

LEFT ALIGNED (Label):
  • No.ID :
  • Tanggal :
  • Produk names
  • Diskon
  • Subtotal
  • metode

RIGHT ALIGNED (Amount):
  • Invoice numbers
  • Prices
  • Totals
  • Payment amounts

CENTER ALIGNED (Header/Footer):
  • Nama toko
  • Alamat toko
  • Pesan terima kasih
  • Separator line

┌─────────────────────────────────────────────────────────────────────────────┐
│                       FORMAT SPECIFICATIONS                                  │
└─────────────────────────────────────────────────────────────────────────────┘

PAPER WIDTH:
  └─ 32 karakter (standard thermal printer 58mm)

SEPARATOR LINE:
  └─ _ (underscore) repeated 32 times
  └─ Automatic calculation, tidak perlu manual

PRICE FORMATTING:
  ✓ 18000 → "18.000" (thousand separator)
  ✓ 1000000 → "1.000.000" (multiple separators)
  ✓ Automatic menggunakan Indonesian locale

DATE FORMATTING:
  ✓ 2026-05-12 12:01 → "12-05-2026,12.01"
  ✓ DD-MM-YYYY,HH.MM format
  ✓ Automatic format conversion

CUSTOMER TYPE:
  ✓ 'member' → display "Member"
  ✓ 'regular' → display "Regular"
  ✓ Conditional fields untuk member-only info

PAYMENT METHOD:
  ✓ 'cash' → "Tunai"
  ✓ 'transfer' → "Transfer"
  ✓ 'debit_card' → "Debit"
  ✓ 'credit_card' → "Kredit"
  ✓ 'qris' → "QRIS"
  ✓ 'e_wallet' → "E-Wallet"

   • Harga auto dengan thousand separator
   • Tanggal auto format Indonesia

MINIMAL EXAMPLE:
  const receipt = {
    invoiceNumber: 'INV-001',
    date: new Date(),
    customerName: 'Pembeli',
    items: [
      { productName: 'Produk', quantity: 1, price: 10000 }
    ],
    subtotal: 10000,
    total: 10000,
    amountPaid: 10000,
    change: 0,
  };

  await printSimpleReceipt(receipt);


WITH DISCOUNT & POINTS:
  const receipt = {
    ...receipt,
    customerType: 'member',
    discount: 1000,
    pointsRedeemed: 2,
    pointsValue: 2000,
    total: 7000,
    earnedPoints: 1,
  };

  await printSimpleReceipt(receipt);


PREVIEW (WITHOUT PRINTING):
  const preview = generateReceiptRaw(receipt);
  console.log(preview);
